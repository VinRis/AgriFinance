# **App Name**: AgriFinance Pro

## Core Features:

- Livestock Selection: Landing page to select livestock type (Dairy, Poultry).
- KPI Dashboard: Display KPIs relevant to the selected livestock enterprise, summarizing financial health. This tool reasons when and how to incorporate specific pieces of information when generating a dashboard. (Not dependent on AI)
- Record Entry: Form to input new records, dynamically adjusting fields based on the selected livestock enterprise.
- Record Management: View, filter (by month), edit, and delete records. Includes bulk delete and CSV export.
- Report Generation: Export professional-looking PDF reports suitable for presentation to banks or investors.
- Settings Panel: Toggle dark/light mode, set farm name, manager name, location, currency, backup, and restore data.
- Data Persistence: Implement offline data persistence for records and settings.

## Style Guidelines:

- Primary color: Forest green (#3F704F) to evoke growth, nature, and prosperity.
- Background color: Light beige (#F5F5DC), a desaturated and bright hue from the same region of the color wheel.
- Accent color: Muted gold (#A78B48) to symbolize wealth and highlight important elements.
- Body and headline font: 'PT Sans' (humanist sans-serif) for a balance of modernity and approachability.
- Use clear, concise icons representing livestock, financial transactions, and reporting functions.
- Clean, organized layout with a bottom navigation bar for main sections. Utilize a floating action button (FAB) for adding new records.
- Subtle transitions and animations to enhance user experience and provide feedback on interactions.
